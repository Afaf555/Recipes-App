# Апликација за прикажување на рецепти користејќи го јавниот API TheMealDB.
Апликацијата ги прикажува категориите на јадења, сите јадења од избрана категорија, детални рецепти и рандом рецепт на денот.
Преземање на сите категории од API:
 https://www.themealdb.com/api/json/v1/1/categories.php
-Приказ на име, слика и краток опис во картички
-Пребарување категории по име или опис
-Клик на категорија → премин кон екранот со јадења

## Јадења по категорија

-Endpoint:
 https://www.themealdb.com/api/json/v1/1/filter.php?c={category}
-Приказ во grid layout
-Слика + име на јадење
-Локално пребарување на јадења во таа категорија

## Детален приказ на рецепт

-Endpoint:
 https://www.themealdb.com/api/json/v1/1/lookup.php?i={id}
-Прикажува:
Слика, име, инструкции, листа на состојки + мерки, YouTube линк (ако постои)

## Рандом рецепт на денот
-Endpoint:
 https://www.themealdb.com/api/json/v1/1/random.php
-Копче во AppBar кое отвора целосен детал за рандом рецепт
